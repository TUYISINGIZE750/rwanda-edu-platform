Enhanced Registration System

Automatic School Display by District Selection

1 Select Your Location

API Endpoint Used:
GET /api/v1/schools-by-district/district/{province}/{district}

2 Schools Auto-Display

Select a province and district to see available TVET/TSS schools
  • ✓ Automatic display when district is selected
  • ✓ Shows school name, type, and category
  • ✓ Displays number of available trades
  • ✓ Preview of available trades

3 Complete Registration Flow

  • → User selects school → System shows all trades in that school
  • → User selects trade → System shows all levels (Level 1-6)
  • → User selects level → Registration complete
